# Encryption Demo
Simple Python Script to encrypt a text message symmetrically and asymmetrically, then decode that same message

**Link to project:** https://github.com/Bakerhewitt/Encryption

## How It's made:
Tech used: Python in Visual Studio Code
Import libraries from cryptography to allow for encryption
Created my message (A reference to Death Note, an anime. A character must puzzle out a secret message)
First I worked on the symmetric encryption as it was more straightforward.
The asymmetric encryption section took longer and I had to rely on Google a bit for key generation code.

## Lessons Learned:
I liked seeing my code in action. Symmetrical encryption seems easier to set up and understand, while Asymmetrical was less intuitive for me.
I had to google some stuff about asymmetrical encryption and refer back to the videos in the course, and I still don't think I understand it completely.
I would like to spend more time on asymmetrical encryption in the future.

## How to run

```
py -m pip install cryptography
py encryption_demo.py
```
